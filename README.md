# Restaurant Sales Performance & Revenue Analysis
## **Overview**
An end-to-end Business Intelligence project in Excel that transforms raw, fragmented restaurant sales data into actionable strategic insights.

---

## **Tools**
- Excel  
- Power Query (M Language)  
- Pivot Tables  
- Conditional Formatting  

---

## **Data Preparation**
The raw dataset contained significant quality issues (documented in `Data_issues worksheet`). I resolved these using **custom M functions** to ensure the analysis remained statistically sound:
* **Automated Imputation:** Handled **10% missing values** in 'Item' names and **4% in 'Pricing'** using median-based logic to avoid skewing revenue totals.
* **Error Handling:** Addressed 5% nulls in payment methods and standardized inconsistent order records.
---

## **Features**
- **Pivot Tables** for revenue and category analysis  
- **2 Interactive Dashboards** with slicers  
- **What-If Scenario** for revenue forecasting  
- **Conditional Formatting** for performance insights
